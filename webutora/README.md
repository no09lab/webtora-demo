# うぇぶとら（Webutora）

Webカメラだけで全身トラッキングを行い、VRChatへOSCで送信するPython製ツールです。

「まず動く」を最優先に、軽量・シンプル・実用性を重視した構成で制作しています。

---

# 概要

うぇぶとらは、Webカメラから取得した映像をMediaPipe Poseで解析し、人体姿勢を推定します。

推定した姿勢情報を3D座標へ変換し、体型補正やキャリブレーションを行ったうえで、VRChatへOSC経由で8点トラッカーとして送信します。

GUIから起動・停止・キャリブレーション・各種調整を行えるため、手軽に利用できます。

---

# 主な機能

- MediaPipe Poseによる全身トラッキング
- 3D座標生成
- 身長・骨格差を補正するキャリブレーション
- VRChat(OSC)への8点トラッカー送信
- トラッカー個別ON/OFF
- 全体ON/OFF
- Tkinter製GUI
- 全体位置のリアルタイム調整
- カウントダウン付き再キャリブレーション

# セットアップ

Python 3.10をインストールしてください。

https://www.python.org/downloads/release/python-3100/

その後、

1. Webutoraフォルダを任意の場所へ配置
2. 初回のみ `setup.bat` を実行
3. `run.bat` を実行
4. VRChat側でOSCを有効化
5. GUIからトラッキング開始
6. 初回キャリブレーションを実行

# 注意事項・制限

本ツールのトラッキング精度は、一般的なフルトラ機材には及びません。

ただし、立った状態での軽い動きやジェスチャーであれば、実用レベルで追従します。

照明条件やWebカメラの設置位置・角度の影響を大きく受けます。

暗所・逆光・身体が隠れる環境では精度が低下する場合があります。

また、本ツールは立位での使用を前提としています。

座る・寝転ぶなどの姿勢では正常なトラッキングは期待できません。
